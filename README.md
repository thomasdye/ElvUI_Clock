# ElvUI Clock
A simple but useful clock for ElvUI

Mouse over to view the controls for the clock

## Left Click
opens/closes Calendar

## Right Click
opens/closes Stopwatch

## Shift + Right Click
changes hour format from 12 to 24 or vice versa
